Project-GC Addon Progress

This page lets you select your "My Finds" GPX file and get a report of your progress towards the Project-GC BadgeGen addons. This report is generated entirely in your browser; no part of your GPX file is sent over the internet. Please be patient if you have a large number of finds. Expand the list below to see some limitations of this tool. This tool requires a modern web browser in order to function correctly.



Limitations
  • Lab caches are completely absent from the "My Finds" GPX file. All addons for the Lab Cacher Badge are therefore excluded. This can also cause the value for the Busy Cacher Badge and Diverse Cacher Badge addon to be one type short.
  • The addons for the Achiever Badge may be slightly incorrect as Project-GC maintains a list of exceptions for classifying a cache as a challenge cache.
  • The addons for the All Around Cacher Badge are excluded for three reasons:
    • If you take advantage of the solved coordinate feature, then the original coordinates will not be listed in the "My Finds" GPX file, and it is the original coordinates that are used by Project-GC for this computation.
    • Your home location is not listed in your "My Finds" GPX file and would have to be entered separately.
    • Project-GC itself recognizes that small differences in how the angle is computed could result in different outcomes in other tools for caches near the angle boundaries.
  • You will only earn credit for a hosted event if you also logged that you attended.
  • The addons for the High Altitude Cacher Badge and Low Altitude Cacher Badge are excluded as elevation information is not present in the "My Finds" GPX file.
  • The addons for the Long-Distance Cacher Badge are excluded as the home coordinates are not present in the "My Finds" GPX file. Since these are either completed or not (i.e. you can't make partial progress on them), I don't deem it important enough to make a way for the home coordinates to be entered separately.
  • The All counties addon is excluded from the Country Badges as this information is not included in the "My Finds" GPX file.
  • I do not know if I correctly handled cases where there are more than one found log on a cache.
  • Grid loops are not reported in the "Way to 81" tables.
  • Version 1.0 GPX files (as opposed to version 1.0.1) do not contain attribute information for caches, which will cause a lot of information to be incorrect, including the tool thinking you do not have any challenge caches. The GPX file version can be changed under Settings → Preferences → GPX Version.
